# Model Equations

## Simple Regression 1

Monthly Sales = 560777.35 + 2.13 × Marketing Spend

Interpretation:

Each additional unit of marketing spend is associated with approximately 2.13 additional units of sales.

## Simple Regression 2

Monthly Sales = 446410.58 + 35.68 × Footfall

Interpretation:

Each additional customer visit is associated with approximately 35.68 additional units of sales.

## Multiple Regression

Monthly Sales = 135425.92 + 1.1767(Marketing Spend) + 33.6613(Footfall) + 2924.49(Inventory Availability %) − 1027.69(Region North)

## Dummy Variable Approach

Categorical Variable Used: Region

Reference Category: East

Dummy Variables Created:

* Region North
* Region South
* Region West

East was used as the reference category to avoid the dummy variable trap.

## Final Model

The multiple regression model was selected because it achieved the highest R-squared value and provided the strongest business insights.
